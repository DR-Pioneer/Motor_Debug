# Motor_Debug
This is a software tool based on FOC/BLDC motor control strategies, which generates corresponding control code according to the chip model and packaging method. It aims to quickly demonstrate motor solutions and provide motor parameter tuning functions.
## OVERVIEW

## Guide
- Drag the Keil project file (.uvprojx/.axf) to the variable area.
- Drag the variable to the window you need, and click the start button to observe the variable.
- The curve window can observe up to 10 variables simultaneously.
- The RTT window supports fast sampling of variables, but only supports one channel.
## Requisites
- install the JLink driver https://www.segger.com/downloads/jlink/.
